# Project-Greenden

Greenden is a responsive multi-page website built using HTML and Tailwind CSS. It was developed as a practice project to strengthen frontend development skills and understand how modern websites are structured using utility-first CSS.

## About the Project

This project focuses on building a clean and responsive user interface using Tailwind CSS. It demonstrates how real-world websites are structured with multiple pages and consistent design principles.

## Pages Included

### Home Page
- Acts as the landing page of the website  
- Features a clean and attractive layout  
- Provides a short introduction to Greenden  

### Products Page
- Displays a collection of plant/green-themed products  
- Organized using a responsive grid/card layout  
- Focuses on clean spacing and visual balance  

### Contact Page
- Shows contact details in a structured layout  
- Designed for simple and clear user interaction  

## Responsive Design
- Fully responsive layout for all screen sizes  
- Optimized for mobiles and desktop devices  
- Built using Tailwind CSS utility classes (`sm:`, `md:`, `lg:`)  
- Ensures consistent spacing, and alignment for devices  

## Tech Stack
- HTML5  
- Tailwind CSS  

## Key Learnings
- Structuring a multi-page website using HTML  
- Using Tailwind CSS utility classes effectively  
- Implementing responsive design with breakpoints  
- Improving layout design and UI consistency  
- Building real-world frontend project structure  

## Project Outcome
By building Greenden, I improved my understanding of responsive web design and gained hands-on experience with Tailwind CSS. This project helped me practice creating modern, clean, and user-friendly web layouts suitable for real-world applications.
